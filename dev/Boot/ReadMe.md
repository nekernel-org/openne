# BootZ (Zeta Bootloader Architecture)

You need:

- MinGW for the Kernel and bootloader.
- Netwide Assembler to output COFF object code.

Start by cloning the repo:

```
git clone git@github.com:amlel-el-mahrouss/zka-dev.git
```

And then execute:

```
make all
```
